The most important tags for SEO:

The most important tags for SEO:

  • <head>: contains information about your website that isn't visible, but very important for SEO
  • <title>: used by search engines to determine the subject of a page
  • <meta>: usually displayed as a small description on search engines
  • Header tags (e.g. <h1> through <h6>): will help search engines understand the content of your webpage. They are also user-friendly!
  • <img>: using the alt attribute with your image is great for screen readers, but also important for search engine crawlers.
  • <a>: very important for web accessibility
  • Semantic tags (e.g. <nav> and <header>): makes it easy for search engines to parse through html files, as semantic tags describle separate page components
  • Canonical (e.g. <link> w/ rel= "canonical"): will let search engines know what the main pages are for a certain topic. Will help combat conflict in ranking between webpages.
  • Open graph (e.g. <meta> w/ property="og:title"): allows your website to become a rich object in social networks.
  • Robots (e.g. <meta> w/ name="robots"): indicates which pages on your website should be indexed for SEO.

Explanations:

  1. Using 10px: An absolute size. Use when you want something to be the same size regardless of device or size of browser.
  2. Using 1em: A relative size, specifically relative to it's parent's size. I set the parent's font size to 12px, so this element is 12px as well.
  3. Using 1rem: Another relative size, specifically relative to the root's size (which is set to 15px). Should use when you want something relative to the root font size.
  4. Using 2em: will be double the size of the parent. It is 24px.
  5. Using 2rem: will be double the size of the root size. It is 30px.
  6. Using 2cm: an absolute size which will be 2cm. Is useful when you want something to be a specific size when it's printed out.
